不同权重的抽奖工具

数据工具 / Python / 实践项目
Python Excel 数据处理 活动工具

项目简介

在 Toujours Flowers 实习期间，我参与策划并执行了一次线上抽奖活动。 与常规抽奖不同，本次活动的中奖概率并非均等， 而是基于用户互动数据分配权重（如参与频率、互动行为等）。

由于市面上常见的抽奖工具仅支持等概率随机， 无法满足“不同用户对应不同中奖权重”的需求， 我使用 Python + Excel 设计并实现了一套 可配置、可复用的加权抽奖工具，用于实际活动落地。

（我必须承认，它还有很多可以完善的地方，从样式美化到代码本身……但这是它在2025年6月最初的样子， 也是第一次毛遂自荐自己的“解决方案”，因此值得记录一下;D）
